Will this get me banned? Not a chance.

BetterVibrance adjusts your monitor's color output through official display APIs — the exact same ones your GPU control panel uses. It has zero interaction with any game.

  • Never modifies game files or configs
  • No DLL injection of any kind
  • Never reads or writes process memory
  • No hooks into running applications
  • Only uses standard Windows display APIs
  • Compatible with EAC, BattlEye, Vanguard, FACEIT
  • Runs as a standalone system tray app
  • Equivalent to changing settings in NVIDIA Control Panel
