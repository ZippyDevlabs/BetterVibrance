# Legal

## Safety & Transparency

BetterVibrance adjusts display color settings (Digital Vibrance, Gamma, Contrast) using **standard Windows display APIs** — the same interfaces used by the NVIDIA Control Panel, AMD Radeon Software, and similar first-party tools.

- **It does not modify, patch, or inject into any game files or processes.**
- **It does not read from or write to game memory.**
- **It does not hook into graphics drivers or rendering pipelines.**
- **It operates entirely at the OS display-settings level.**

BetterVibrance is fully compatible with all major anti-cheat systems, including but not limited to EasyAntiCheat (EAC), Vanguard, BattlEye, and FACEIT Anti-Cheat. Adjusting your monitor's color output through Windows APIs is a normal, permitted operation — identical to changing settings in your GPU control panel.
