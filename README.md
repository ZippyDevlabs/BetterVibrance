# BetterVibrance

**BetterVibrance** is a lightweight Windows system tray application that lets you instantly switch between display color profiles — optimized for gaming, content creation, or everyday use.

Configure your Digital Vibrance, Gamma, and Contrast, choose which monitors the app should control, and switch between profiles with a single keypress.

---

## Features

- **Monitor selection** — choose which monitors BetterVibrance should control; profiles apply to all selected monitors
- **Instant profile switching** — switch between profiles using fully customizable global hotkeys
- **Game-aware switching** — automatically apply a profile when a specific game or app launches, and restore your default when it closes
- **Cycle mode** — step through multiple profiles in sequence with a single hotkey
- **System tray** — runs silently in the background, zero clutter
- **Start with Windows** — always ready when you are

---

## Who Is It For?

- **Gamers** who want better visbility in-game but accurate colors for everything else
- **Streamers and content creators** who switch between a calibrated editing profile and a high-vibrance viewing profile
- **Multi-monitor users** who want to control which displays get color-enhanced

---

## Requirements

- Windows 10 / 11 (64-bit)
- NVIDIA GPU
- NVIDIA drivers installed

---

## Getting Started

1. Download the latest installer from the [Releases](../../releases) page
2. Run the installer and launch BetterVibrance
3. Enter your license key when prompted
4. Open Settings to configure your monitors and profiles
5. Assign your preferred hotkeys and switch profiles instantly
